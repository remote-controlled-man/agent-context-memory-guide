# Agent Memory Learn

一个独立于现有 `sites/agent-memory-guide` 的纯静态学习站。

## 目标

- 面向没有计算机背景的读者解释 6 个 agent 项目的 `context` 和 `memory`
- 所有关键结论都能回到 `docs/` 中的证据入口
- 构建结果是 GitHub Pages 可直接部署的纯静态文件

## 构建

```bash
npm run build
```

构建产物输出到 `dist/`。

## 页面

- `index.html`：首页
- `glossary.html`：术语页
- `compare.html`：横向对照页
- `patterns.html`：共同规律页
- `frameworks.html`：框架选择页
- `blueprint.html`：实现蓝图页
- `repos/*.html`：6 个 repo 学习页
- `sources/*.html`：`docs/` 对应的证据页
