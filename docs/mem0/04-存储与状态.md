# `mem0`：存储与状态

## 1. 默认目录

`MEM0_DIR` 和默认 `history_db_path` 定义在配置层；`memory/setup.py` 会创建 `~/.mem0` 和 `config.json`。这套目录是 Python OSS 核心的本地状态根。

### 源码锚点

- `mem0/configs/base.py:13-13`
- `mem0/memory/setup.py:12-12`

## 2. SQLite 历史与消息窗口

`storage.py` 维护两类表：

- `history`：记录 add/update/delete 等历史动作
- `messages`：保存最近消息窗口

它们属于运行时辅助状态，不是长期 memory 本体。

### 源码锚点

- `mem0/memory/storage.py:11-257`

## 3. 向量库与路径

向量库后端由 `VectorStoreFactory` 和各 provider config 决定，本地文件型 provider 会补默认路径。长期 memory 本体主要驻留在向量库，而不是 SQLite。

### 源码锚点

- `mem0/utils/factory.py:30-30`
- `mem0/vector_stores/configs.py:6-6`
- `mem0/vector_stores/faiss.py:131-131`

## 4. entity collection

除了 memory 主 collection 之外，`entity_store` 还维护独立实体集合，用于把相关 memories 聚合到实体层。

### 源码锚点

- `mem0/memory/main.py:327-327`
